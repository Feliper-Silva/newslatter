## :rocket: Utilizar

Siga os passo-a-passo:

```bash
# clone o projeto:
$ git clone https://github.com.br/feliper-silva/newslatter
# entre na pasta do projeto:
$ cd newslatter
```

```bash
# Instale as dependências:
$ yarn
# Inicializar
$ yarn start
# Lembrando que se usa npm utilize os comandos do npm.
```
